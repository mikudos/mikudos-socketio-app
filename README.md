# MIKUDOS SOCKETIO SERVER

mikudos-socketio-server.

## Usage

Import the mikudos-socketio-server module:

```ts
import {
    Application,
    Authentication,
    AuthenticationRequest,
    RpcServiceMethods,
    JSON_RPC_HANDLER,
    CHAT_HANDLER,
    DUPLEX_HANDLER
} from 'mikudos-socketio-server';
```

```js
var {
    Application,
    Authentication,
    AuthenticationRequest,
    RpcServiceMethods,
    JSON_RPC_HANDLER,
    CHAT_HANDLER,
    DUPLEX_HANDLER
} = require('mikudos-socketio-server');
```
