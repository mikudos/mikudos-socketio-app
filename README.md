# MIKUDOS SOCKETIO SERVER

mikudos-socketio-server.

## Usage

Import the mikudos-socketio-server module:

```ts
import {
    Application,
    ServiceMethods,
    JSON_RPC_HANDLER,
    ChatHandler
} from 'mikudos-socketio-server';
```

```js
var {
    Application,
    ServiceMethods,
    JSON_RPC_HANDLER,
    ChatHandler
} = require('mikudos-socketio-server');
```
